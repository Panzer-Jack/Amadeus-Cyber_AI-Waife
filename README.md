# Amadeus-Cyber_AI-Waife
仿 Amadeus 的 一个 玩具（）， 手机版的 Cyber_AI-Waife
Cyber_AI-Waife  For IOS or Android

- 登陆界面
- 主菜单界面
- 电话模式 
- 视频模式（ H5 + Live2D ）
